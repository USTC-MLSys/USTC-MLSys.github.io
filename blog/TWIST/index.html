TWIST

This page collects figures related to DHeLlam, an ICCD 2025 Best Paper on general-purpose automatic micro-batch co-execution for distributed LLM training. The current note focuses on the available TWIST visuals, including the overview figure, layer breakdown, and end-to-end MFU comparison.

Scope

This page is a compact technical note built around the TWIST figures already available in the repository and the corresponding DHeLlam publication.

What can be confirmed from the available figures

  • The note includes three figures: an overview, a layer breakdown, and an MFU comparison.
  • The MFU comparison figure is titled "End-to-End MFU Comparison on H200 (Qwen3-14B-like)".
  • The MFU figure compares Megatron-LM and TWIST under two configurations: TP=4, PP=2 and TP=8, PP=1.
  • The figure labels show 46% vs 49% for one configuration and 41% vs 47% for the other.
