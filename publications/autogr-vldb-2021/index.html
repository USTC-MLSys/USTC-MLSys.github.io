AutoGR: Automated Geo-Replication with Fast System Performance and Preserved Application Semantics

AutoGR proposes an end-to-end geo-replication deployment framework to free programmers from label-intensive tasks. It automatically enables geo-replication features for non-replicated, serializable applications with optimized performance and correct application semantics, building upon the Olisipo fine-grained coordination framework.

Authors

Abstract

AutoGR automates geo-replication deployment by inferring required consistency annotations, enabling fast system performance and correct application semantics without manual programmer effort.
