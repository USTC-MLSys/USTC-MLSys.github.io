Publication

FrozenHot Cache: Rethinking Cache Management for Modern Hardware

Inspired by highly skewed data popularity and short-term hotspot stability in cache workloads, FrozenHot splits cache space into a frozen cache and a dynamic cache. The frozen cache serves hot objects with minimal latency by eliminating cache management locks, while the dynamic cache uses existing designs for workload adaptability.

EuroSys 2023 / May 2023

Authors

Ziyue Qiu, Juncheng Yang, Juncheng Zhang, Cheng Li, Xiaosong Ma, Qi Chen, Mao Yang, Yinlong Xu

Abstract

FrozenHot divides cache into a frozen segment (for hot objects, lock-free) and a dynamic segment (for adaptability), significantly improving cache throughput and scalability on modern multi-core hardware.
