Publication

Lunule: An Agile and Judicious Metadata Load Balancer for CephFS

Lunule is a novel CephFS metadata load balancer that employs an imbalance factor model for accurately determining when to trigger rebalancing and tolerate benign imbalanced situations. It dynamically adjusts migration plans including migration amount determination and subtree selection based on different workloads.

Abstract

Lunule introduces an imbalance factor model for CephFS metadata load balancing that accurately determines rebalance triggers and dynamically adapts migration plans to workload characteristics.
